# Immaterial-shell
A robust and extensive desktop shell built with quickshell for hyprland. 
Not exactly 'feature-complete' yet.
Dependencies:
```
matugen
any nerd font (Google Space Mono is a good one)
Google Sans Flex font
```
## Usage
Make sure you have `qs` in your exec-once
Run `qs ipc show` in your terminal for a grasp of keybind/keychord mappings.
As of now, there's not a built-in wallpaper selector.
