# Immaterial-shell
A robust and extensive desktop shell built with quickshell for hyprland. \n
Not exactly 'feature-complete' yet. \n
Dependencies:
```
matugen
any nerd font (Google Space Mono is a good one)
Google Sans Flex font
```
## Usage
Make sure you have `qs` in your exec-once \n
Run `qs ipc show` in your terminal for a grasp of keybind/keychord mappings. \n
As of now, there's not a built-in wallpaper selector.
